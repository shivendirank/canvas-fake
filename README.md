A responsive Canvas-inspired dashboard that works on desktop and mobile.

## Run Locally

Option 1 (VS Code Live Server extension):
1. Open `index.html`
2. Start Live Server

Option 2 (Python simple server):
1. In this folder run:
   ```bash
   python -m http.server 8080
   ```
2. Open `http://localhost:8080`

## Deploy (Phone + Website)

### Netlify (easiest)
1. Go to Netlify and create a new site.
2. Drag and drop this project folder (or connect a Git repo).
3. Your site is instantly hosted over HTTPS.

### Vercel
1. Push this folder to GitHub.
2. Import the repo in Vercel.
3. Deploy with default settings.

After deployment, open the URL on desktop and phone. The layout automatically adapts for mobile screens.
